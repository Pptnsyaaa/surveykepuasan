A. FRONTEND (ANTARMUKA PENGGUNA)
------------------------------------------------------------------------

1. File: `frontend/src/pages/admin/DashboardAdmin.jsx`
   [Modifikasi & Perbaikan]
   - Perbaikan Bug JSON Parse: Menulis ulang logika di dalam `fetchSurvey()`. Sebelumnya aplikasi mati total (White Screen) saat menerima data string JSON yang rusak pada kolom `responses` atau `student`. Ditambahkan validasi `typeof === "string"` dan pembungkus `try/catch`.
   - Penyesuaian UX Refresh Rate: Mengubah nilai fungsi `setInterval()` pada akhir komponen dari `3000` (3 detik) menjadi `30000` (30 detik). Ini mencegah layar berkedip terus-menerus dan meringankan beban API.
   - Penutupan Celah Keamanan Fatal: Menyuntikkan fitur "Route Guard" di baris paling atas komponen menggunakan `useEffect()`. Logika ini akan memeriksa `localStorage.getItem('adminLogin')`. Jika tidak ada, sistem akan langsung menendang penyusup dengan peringatan alert() dan perintah `window.location.href = '/admin/login'`.

2. File: `frontend/src/components/dashboard/TrenWaktu.jsx`
   [Modifikasi & Perbaikan]
   - Penghapusan Library Bermasalah: Menghapus total fungsi pemanggilan modul `import { motion } from 'framer-motion'` beserta elemen pembungkus `<motion.div>` yang sebelumnya membuat aplikasi error.
   - Perbaikan Siklus Chart.js: Menambahkan logika `.destroy()` pada `useEffect()` untuk memastikan grafik kanvas yang lama dihancurkan sebelum merender grafik baru, mencegah error penumpukan kanvas (Canvas is already in use).
   - Penstabilan Resolusi Container Grafik: Mengganti kelas CSS pembungkus grafik dari yang bersifat melar dinamis (`h-[250px] sm:h-[300px] md:h-[400px]`) menjadi absolut dan paten (`h-[300px]`). Hal ini menghentikan pemuatan ulang Chart.js di detik pertama yang mematikan/menggagalkan animasi kemunculan grafik.
   - Injeksi Animasi Eksplisit: Khusus pada blok pengaturan (options) untuk diagram garis (Trend Line Chart), ditambahkan perintah animasi terjun bebas `animation: { y: { duration: 1500, from: 500 }, x: { duration: 1500, from: 0 } }` bertipe 'easeOutQuart' agar pergerakan grafik terlihat sangat jelas walau sebagian besar datanya bernilai nol (0).
   - Perbaikan Scroll Perangkat Sentuh (Mobile): Menambahkan perintah CSS Tailwind `touch-pan-y` beserta inline-style `touchAction: 'pan-y'` di atas tag `<canvas>` agar jari pengguna di layar sentuh tidak tersangkut/macet saat mencoba menggeser layar ke bawah melewati grafik.

3. File: `frontend/src/components/dashboard/DataResponden.jsx`
